## 4. 코드 미리보기 (우측 패널)

```
┌─────────────────────────────────┐
│  📝 DSL 코드                    │
│  [복사📋] [다운로드💾] [닫기✕] │
├─────────────────────────────────┤
│                                 │
│  flowchart LR                   │
│    node_1([Genesis Block])      │
│    node_2[Block #1]             │
│                                 │
│    node_1 --> node_2            │
│                                 │
│  @animation                     │
│    step 1: show node_1          │
│      duration: 1.5s             │
│      effect: fadeIn             │
│  @end                           │
│                                 │
│                                 │
│  [실시간 업데이트 중...]        │
│                                 │
│  마지막 저장: 방금 전            │
│                                 │
│  ☑ 자동 포맷팅                  │
│  ☑ 구문 하이라이팅              │
│                                 │
└─────────────────────────────────┘
```

### 구문 하이라이팅

```
flowchart LR              ← 파란색 (키워드)
  node_1([Genesis Block]) ← 초록색 (노드 ID)
                            검은색 (레이블)
  
@animation                ← 보라색 (섹션)
  step 1: show node_1     ← 주황색 (액션)
    duration: 1.5s        ← 회색 (속성)
@end
```

### 코드 에디터 기능

```
┌─────────────────────────────────┐
│  1  flowchart LR                │ ← 줄 번호
│  2    node_1([Genesis Block])   │
│  3    node_2[Block #1]          │
│  4                              │
│  5    node_1 --> node_2         │
│  6                              │
│  7  @animation                  │
│  8    step 1: show node_1       │ ← 현재 커서 위치
│  9      duration: 1.5s          │
│ 10  @end                        │
│                                 │
│ ⚠️ 구문 오류: 10번째 줄          │
│    '@end' 누락                  │
└─────────────────────────────────┘
```

├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ⏮️ ◀️ ▶️ ⏸️ ⏭️  ━━━━━━●━━━━━━━━━━━  [0:05 / 0:20]          │

│                                                                 │
│  타임라인                                                        │
│  ├─ 0s ─────── 5s ─────── 10s ────── 15s ────── 20s           │
│  │                                                             │
│  │  Step 1 ████                                               │
│  │         node_1 fadeIn (1.5s)                               │
│  │                                                             │
│  │  Step 2      ████                                          │
│  │              highlight (1s)                                │
│  │                                                             │
│  │  Step 3           ████████                                 │
│  │                   connect particles (2s)                   │
│  │                                                             │

│                                                                 │

│  │ 액션: connect                                         │    │
│  │ 대상: node_1 → node_2                                 │    │

│  │ 속도: [2s ━━━●━━━━ 5s]                              │    │
│  │ 색상: [#2196F3 🎨]                                   │    │
│  │                                                       │    │
│  │ 시작 시간: [5.0s]  [◀ 앞으로 | 뒤로 ▶]              │    │
│  │                                                       │    │
│  │ [삭제] [복제] [적용]                                  │    │
│  └───────────────────────────────────────────────────────┘    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 타임라인 인터랙션

#### 스텝 드래그로 순서 변경
```
Before:
Step 1 ████
Step 2      ████
Step 3           ████

드래그하여 Step 3를 Step 1 위치로:

After:
Step 3 ████              ← 순서 바뀜
Step 1      ████
Step 2           ████
```

#### 스텝 길이 조정
```
Step 1 ████ ← 오른쪽 끝을 드래그
       ↓
Step 1 ████████ ← duration 증가
```

#### 스텝 추가
```
┌─────────────────────────┐
│ + 새 애니메이션 스텝 추가│
├─────────────────────────┤
│                         │
│ 액션 선택:               │
│ ( ) Show                │
│ ( ) Hide                │
│ ( ) Highlight           │
│ (●) Connect             │
│ ( ) Camera              │
│                         │
│ 대상:                    │
│ 시작: [node_1 ▾]        │
│ 끝:   [node_2 ▾]        │
│                         │
│ 효과:                    │
│ (●) particles           │
│ ( ) dash                │
│ ( ) glow                │
│                         │
│ 삽입 위치:               │
│ ( ) 맨 앞               │
│ (●) Step 2 이후         │
│ ( ) 맨 뒤               │
│                         │
│    [취소] [추가]        │
